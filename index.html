CentralReport

Monitor your server or your computer very easily!

Download this project as a .zip file Download this project as a tar.gz file

CentralReport meets a simple need: ensuring the health of your computer. Continuously.

Using a simple browser with the IP address of your machine, you can view your real-time statistics:

  • CPU
  • Physical Memory (RAM)
  • Virtual Memory (Swap)
  • Load Average
  • Uptime
  • Disks

CentralReport works on Debian, Ubuntu, CentOS and Mac OS X. A Windows version is planned.

It's developed by three students: @Guimove, @Ninir and @miniche. It's only the beginning: we have many ideas for CentralReport and others exciting and related projets.

You can checkout our blog to read more about the project and the team: blog.centralreport.net. And don't forget to follow us on Twitter: @Central_Report!

Install CentralReport

Our goal? Making it powerful and easy. To install CentralReport, one line a terminal is enough! Please visit the official documentation to get the procedure depending on your system.

Current status

We are working on a cloud solution: you will be able to view all yours computers and servers, and set some email or push alerts.
